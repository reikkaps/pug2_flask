## Installation

### Voraussetzungen

1. python (2.7 oder 3.x)
2. virtualenv

`apt-get install python3 virtualenv`

### Einrichten

1. Virtuelle Umgebung anlegen
   `$> virtualenv -p /usr/bin/python3 venv`

2. Ins Virtualenv wechseln ...
   `$> . venv/bin/activate`

3. Flask installieren und einrichten
   `$ [venv] > pip install Flask`
